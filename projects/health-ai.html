AI-Driven Health Diagnosis

Overview

Medical image classification system designed to detect brain tumors, lung cancer, and pneumonia using deep learning.

Problem Statement

Medical image diagnosis is time-consuming and requires expert review. This project focused on automating disease classification to assist diagnostic workflows.

Approach & Architecture

  • Used pretrained EfficientNet with transfer learning
  • Applied OpenCV-based preprocessing and normalization
  • Implemented data augmentation to reduce overfitting
  • Built a full training and evaluation pipeline

Results & Impact

  • High-accuracy classification across 3 disease classes
  • Reduced overfitting by 25%
  • Improved robustness on unseen medical images

Tech Stack

TensorFlow, EfficientNet, OpenCV, CNNs, Medical Image Processing
